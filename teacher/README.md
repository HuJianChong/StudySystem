# 教师端教育系统原型

这是一个高保真的教育系统原型设计，专为教师端平板应用（16:9尺寸）设计。

## 主要功能

本原型包含四个主要模块：

1. **章节管理**：增删改查功能
2. **课程管理**：详情、新增、修改、生成试题、删除等功能
3. **教材管理**：新增、列表、导入功能
4. **试题管理**：列表、详情查看、考试学生及成绩查看

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome 图标
- 纯JavaScript交互（无框架依赖）

## 文件结构

- `index.html` - 主入口文件，包含侧边栏导航和内容区域的iframe
- `chapters.html` - 章节管理页面
- `courses.html` - 课程管理页面
- `materials.html` - 教材管理页面
- `exams.html` - 试题管理页面
- `styles.css` - 自定义样式

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 通过侧边栏导航切换不同的功能模块
3. 点击各种按钮可以查看交互效果（模态框等）

## 原型说明

- 原型使用了响应式设计，适合在平板设备上展示
- 所有页面都经过优化，以确保16:9的屏幕比例下有最佳显示效果
- 界面设计遵循现代UI设计规范，注重用户体验

## 注意事项

- 这是一个纯前端原型，不包含后端功能
- 部分交互功能仅作展示用途，如模态框和下拉菜单
- 图片来源于Unsplash，可用于非商业用途

## 适配Figma

- 原型设计考虑了后续转为Figma文件进行二次编辑的需求
- 使用了清晰的组件结构和一致的设计语言
- 样式主要使用Tailwind CSS实现，便于在设计工具中还原
